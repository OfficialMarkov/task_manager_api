# Руководство по настройке проекта

## Ответы на вопросы

### 1. Где создать .env файл и какие переменные поменять?

**Где создать:**
- Файл `.env` нужно создать в **корне проекта** (там же, где находится `env.example`)
- Путь: `d:\Programmig\project\task_manager_api\task_manager_api\.env`

**Как создать:**
1. Скопируйте файл `env.example` и переименуйте в `.env`
2. Или создайте новый файл `.env` в корне проекта

**Какие переменные ОБЯЗАТЕЛЬНО нужно поменять:**

```env
# 1. SECRET_KEY - ОБЯЗАТЕЛЬНО измените на случайную строку!
# Генерируйте длинный случайный ключ (минимум 32 символа)
# Можно использовать: python -c "import secrets; print(secrets.token_urlsafe(32))"
SECRET_KEY=ваш-случайный-секретный-ключ-минимум-32-символа

# 2. DATABASE_URL - настройте под вашу БД
# Формат: postgresql+asyncpg://пользователь:пароль@хост:порт/имя_бд
DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/task_manager

# 3. Если используете Docker - настройте эти переменные:
POSTGRES_USER=postgres          # Имя пользователя PostgreSQL
POSTGRES_PASSWORD=postgres      # Пароль PostgreSQL (измените!)
POSTGRES_DB=task_manager        # Имя базы данных
```

**Переменные, которые можно оставить как есть (если подходят):**
- `ACCESS_TOKEN_EXPIRE_MINUTES=30` - время жизни токена (30 минут)
- `DEFAULT_PAGE_SIZE=10` - размер страницы по умолчанию
- `MAX_PAGE_SIZE=100` - максимальный размер страницы

### 2. Про какие зависимости идет речь?

**Зависимости** - это библиотеки Python, которые нужны для работы проекта.

**Где они указаны:**
- В файле `pyproject.toml` в секции `[project]` -> `dependencies`

**Основные зависимости:**
1. **fastapi** - веб-фреймворк для создания API
2. **uvicorn** - сервер для запуска FastAPI
3. **sqlalchemy** - ORM для работы с БД
4. **asyncpg** - драйвер для асинхронной работы с PostgreSQL
5. **alembic** - система миграций БД
6. **pydantic** - валидация данных
7. **python-jose** - работа с JWT токенами
8. **passlib** - хеширование паролей
9. **psycopg2-binary** - синхронный драйвер PostgreSQL (для Alembic)

**Как установить:**
```bash
# Установка всех зависимостей
pip install -e .

# Или с dev зависимостями (включая pytest)
pip install -e ".[dev]"
```

**Где используются:**
- При запуске приложения Python автоматически загружает эти библиотеки
- Если библиотеки не установлены - будет ошибка `ModuleNotFoundError`

### 3. Проверка работоспособности

Проект проверен на следующие ситуации:

✅ **Обработаны:**
- Отсутствие .env файла (будет ошибка при запуске - это нормально)
- Неверные данные в запросе (валидация Pydantic)
- Неавторизованные запросы (401 ошибка)
- Недостаточно прав (403 ошибка)
- Несуществующие ресурсы (404 ошибка)
- Ошибки БД (500 ошибка с общим сообщением)
- Пустые пароли (проверка в security.py)
- Дублирование email (проверка при регистрации)

✅ **Добавлены комментарии:**
- Каждый файл имеет подробные комментарии на русском языке
- Каждая функция объяснена простым языком
- Описано, как работает код и зачем он нужен

## Структура .env файла

```env
# ============================================
# БАЗА ДАННЫХ
# ============================================
# URL подключения к PostgreSQL
# Формат: postgresql+asyncpg://пользователь:пароль@хост:порт/база_данных
DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/task_manager

# Настройки PostgreSQL (для Docker)
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres          # ИЗМЕНИТЕ В PRODUCTION!
POSTGRES_DB=task_manager
POSTGRES_PORT=5432

# ============================================
# БЕЗОПАСНОСТЬ (JWT)
# ============================================
# Секретный ключ для подписи JWT токенов
# ОБЯЗАТЕЛЬНО измените на случайную строку!
# Генерация: python -c "import secrets; print(secrets.token_urlsafe(32))"
SECRET_KEY=your-secret-key-here-change-in-production

# Алгоритм шифрования JWT (обычно не меняют)
ALGORITHM=HS256

# Время жизни токена в минутах
ACCESS_TOKEN_EXPIRE_MINUTES=30

# ============================================
# API НАСТРОЙКИ
# ============================================
# Порт API (для Docker)
API_PORT=8000

# Название проекта
PROJECT_NAME=Task Manager API

# Префикс для всех API эндпоинтов
API_V1_PREFIX=/api/v1

# ============================================
# ПАГИНАЦИЯ
# ============================================
# Размер страницы по умолчанию
DEFAULT_PAGE_SIZE=10

# Максимальный размер страницы
MAX_PAGE_SIZE=100
```

## Частые проблемы

### Проблема: ModuleNotFoundError
**Решение:** Установите зависимости: `pip install -e .`

### Проблема: Ошибка подключения к БД
**Решение:** 
- Проверьте, что PostgreSQL запущен
- Проверьте DATABASE_URL в .env
- Проверьте логи: `docker-compose logs db`

### Проблема: SECRET_KEY не установлен
**Решение:** Добавьте SECRET_KEY в .env файл

### Проблема: Миграции не применены
**Решение:** Выполните `alembic upgrade head`
